# Linear Convection

$$\begin{equation}
\begin{align*}
\frac{\partial u}{\partial t} + c\frac{\partial u}{\partial x} &= 0 \text{, } -\infty < x < \infty \text{, } t > 0 \text{, } c > 0 \\
u(x, 0) &= F(x) \text{, } -\infty < x < \infty \text{, } F(x) \text{: IC function} \\
\implies u(x, t) &= F(x - ct) \text{(Exact solution)}
\end{align*}
\end{equation}$$
